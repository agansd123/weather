## 天气查询

### 需求分析
我们对天气接口的使用需求较为简单，主要为以下两个：

- 按地名查询实时天气；
- 获取最近的天气预报。

> 高德开放平台提供的天气接口满足以上需求。

### 功能设计
虽然我们提供的主要功能为上面两个，但是涉及到编码的层面我们需要考虑的将会更细化一些：

- **主要业务**，获取天气信息；
- **异常处理**，提供统一的异常类，方便调用方定位异常；
- **自定义参数**，需要支持自定义参数，比如超时时间等；
- **提供 Laravel Service Provider**，为 Laravel 框架提供专属封装的入口。
